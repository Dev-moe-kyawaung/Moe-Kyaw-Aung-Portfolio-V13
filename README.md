## 🎯 Overview

**Moe Kyaw Aung Portfolio V13** is a production-ready, fully immersive single-page portfolio website built with pure HTML5, CSS3, and Vanilla JavaScript — zero dependencies, zero frameworks, zero build tools required.

Designed with a **dark parallax aesthetic** using a charcoal (`#222831`) / gunmetal (`#393E46`) / warm khaki (`#948979`) / sand (`#DFD0B8`) palette, it showcases nearly **12 years of Android development expertise** across 17 interactive sections.

> ✨ **Single file · Offline-capable · Mobile-first · WCAG accessible · SEO optimised**

## ✨ Features

### 🎨 Design & UI
- **Parallax / Immersive** scroll-driven animations with layered depth
- **Animated particle canvas** hero background with interactive connecting lines
- **Masonry / Pinterest-style** grid layout for projects and certificates
- **Custom cursor** with magnetic ring effect
- **Dark / Light mode** toggle with smooth transition
- **Preloader** animation with progress bar

### 🔧 Interactive Components
| Feature | Description |
|---------|-------------|
| 🖱️ Custom Cursor | Dot + ring with hover scale effect |
| 📱 Hamburger Menu | Responsive mobile sidebar navigation |
| 🌓 Theme Toggle | Dark ↔ Light mode with CSS custom properties |
| 🎯 Smooth Scroll | Native CSS scroll-behavior + JS fallback |
| 📊 Counter Animation | Scroll-triggered animated statistics |
| 🎠 Carousel | Auto-play testimonials with dot navigation |
| 🔍 Lightbox | Click-to-expand image gallery |
| ❓ FAQ Accordion | Animated expand/collapse with ARIA support |
| ✅ Contact Form | Full client-side validation + success state |
| 📧 Newsletter | Email subscription with confirmation |
| ⬆️ Back To Top | Appears after 400px scroll |
| 📌 Sticky CTA | Persistent hire-me button |
| 🗺️ Google Maps | Embedded location map |
| 📐 Skill Bars | IntersectionObserver-triggered fill animation |
| 🌊 Parallax | Hero content depth scroll effect |
| 🎭 Fade Animations | up / left / right with configurable delay |

### 📱 Responsive
- ✅ Desktop (1200px+) — Full sidebar + multi-column layouts
- ✅ Tablet (768–1100px) — Collapsed sidebar, 2-column grids
- ✅ Mobile (< 768px) — Hamburger menu, single column, touch-optimised

## 🚀 Getting Started

### Option 1 — Direct Open (Fastest)
```bash
# Clone the repo
git clone https://github.com/Dev-moe-kyawaung/Moe-Kyaw-Aung-Portfolio-V13.git

# Open in browser — no server needed
open index.html
# or on Windows:
start index.html
```

### Option 2 — Local Server (Recommended for development)
```bash
# Python 3
python -m http.server 3000

# Node.js (npx)
npx serve .

# VS Code — use Live Server extension
```
Then visit → `http://localhost:3000`

### Option 3 — GitHub Pages (Production)
Push to `main` branch → GitHub Pages auto-deploys via the included workflow.

## 🌍 Deployment

### GitHub Pages (Auto)

The included `.github/workflows/deploy.yml` automatically deploys to GitHub Pages on every push to `main`.

**Manual setup:**
1. Go to repo **Settings → Pages**
2. Source: `Deploy from a branch`
3. Branch: `main` → `/ (root)`
4. Save → your site is live at `https://dev-moe-kyawaung.github.io/Moe-Kyaw-Aung-Portfolio-V13/`

### Other Platforms

| Platform | Command / Method |
|----------|-----------------|
| **Netlify** | Drag & drop `index.html` |
| **Vercel** | `vercel --prod` |
| **Cloudflare Pages** | Connect GitHub repo |
| **Surge.sh** | `surge . your-domain.surge.sh` |
